Class :- In HTML class are not unique and can be used fr different element. An element can have multiple class.

Difference between em and strong tag

Tags em and strong are the tags that adds symantic value to the content.em(emphasize) tag is similar to i(italic) tag. strong tag is similar to b(bold) tag. Both the tags looks similar on webpage visualiy but the difference is used by search engine optimization.

What is attribute?

Attributes is a special keyword that are used to provide additonal information about the element and used as modifier of an element and it adjusts the behaviour of an HTML. It is case sensitive.

Difference between HTML elements and HTML tags

HTML elements :- Elements consists of opening tag, closing tag and content. Self closing tags doesn't come under elements.
HTML tags :- Tags consists of only opening and closing tag.

What is br tag?

br tag :- br tag is used to break the line. Content that are written after br tag starts from next line.
